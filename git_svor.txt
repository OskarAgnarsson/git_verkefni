Git spurningar og svör
1. Hvað gera eftirfarandi Linux skipanir?
	cd: breytir um directory
	cd ~ : breytir í home directory
	ls: sýnir lista af skrám í directory
	pwd: sýnir directory sem þú ert í
	mkdir: býr til directory

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone: klónar repository yfir í tölvu
	git log: sýnir öll commit
	git status: sýnir status á skrám sem þú ert að vinna á
	git diff: sýnir breytingar á skrám sem er ekki búið að setja í staging area
	git checkout: breytir um branch

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff : sýnir breytingar á skrám sem er ekki búið að setja í staging area
	b) git diff --staged : sýnir breytingar á skrám sem er búið að setja í staging area
	c) git diff commit1 commit2: sýnir mun á sérstökum committum

4. 	Hvað er útgáfustýring (e.version control)? version control er log sem leyfir þér að fara til baka og skipta um version

5.	Hverjir eru helstu kostir við að nota GIT? Þú getur alltaf farið til baka um version ef þú rústar öllu og þú getur unnið við það hvar sem er

6.	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu? Strax og þú ert búinn að vinna á einhverjum hlut, eins og eftir hverja spurningu í þessu. Þá getur þú farið til baka ef það virkar ekki rétt

7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT? Working directory er staðurinn sem þú ert að vinna í tölvunni. Staging area inniheldur breyttar skrár áður en þær eru committaðar. Repository er geymsla á öllum skrám í tilteknu verkefni.

8.	Hvenær er sniðugt að nota greinar (branches)? Þegar maður er að vinna á einhverju en vill ekki setja það í master strax

9.	Hver fann upp GIT? Linus Torvalds

10.	Hvar er GIT helst notað? Forritun og vefhönnun
